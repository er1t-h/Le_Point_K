Pourquoi linux est mieux que windows ?

Linux, c'est gratuit

Contrairement à windows, la très grande majorité des distributions linux sont gratuites.

Il n'y a pas qu'un seul linux

En effet, contrairement a windows, il y a beaucoup de choix et de diversités de distributions linux, les plus connues sont Ubuntu, Debian, Arch, OpenSuse, Puppy, Fedora ...

Linux est open-source

Toujours contrairement à windows, le noyau de linux est open-source, cela veut dire que toutes les personnes qui le veulent peuvent acceder au code source, et peut y trouver les bugs, des failles, pour les corriger, tout le monde peut améliorer linux.

Linux est plus stable et fiable que windows

Linux est plus rapide que windows

Linux est plus léger que windows

Linux est totalement customisable

Linux dispose d'une palette d'aplications libres et open-sources

Linux ne demande pas beaucoup de performances

Linux a une excellente ligne de commande

Linux est très facilement portable

Meme vos téléphones sont sous linux

La majorité des serveurs sont sous linux

Il y a un support en ligne completement gratuit

Linux dispose de plusieurs niveaux d'executions superposés

On peut aussi jouer sous linux

Linux a une meilleure gestion des mises a jour

Linux ne vous espionne pas

Linux est bien plus sécurisé que Windows

Linux n'est pas si compliqué à utiliser que cela

Bon, après, windows a quand meme certains avantages.
